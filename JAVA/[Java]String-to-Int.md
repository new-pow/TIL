# [Java] String 을 정수로 타입변환하는 방법

## Integer.parseInt()
- 문자열 파싱하여 int객체 생성

```java
String str = "1122";
int num = Integer.parseInt(str);
```

## Integer.valueOf()
- 문자열로 Integer 객체 생성
```java
String str = "1122";
Integer num = Integer.valueOf(str);
```
