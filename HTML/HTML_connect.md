# HTML: 문서와 연결하기

## HTML - CSS

1. Inline

```html
<p style="color:blue">text</p>
```

2. Internal

```html
<style>
  p {
    color: blue;
  }
</style>
```

3. External / Linked✨

```html
<link rel="stylesheet" href="style.css" />
```

---

## HTML - jQuery

1. CDN 이용하기

```html
<script src="//code.jquery.com/jquery-3.6.0.min.js"></script>
```

2. 다운 받은 파일 연결

```html
<script src="path/jquery-3.6.0.min.js"></script>
```
