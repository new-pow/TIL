# DML

## INSERT

### INSERT Syntax

- multiple data

```sql
INSERT INTO table_name (column1, column2 ...)
VALUES (value1, value2 ....)
        (value1, value2 ....)
        (value1, value2 ....);
```

## UPDATE

### UPDATE Syntax

```sql
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
```

## DELETE

### DELETE Syntax

```sql
DELETE FROM table_name
WHERE condition;
```
