## 작성 규칙

---

- 폴더와 파일명은 영문으로 작성한다.
- 이해한 것을 중심으로 직접 작성한다.
- 부족한 문서는 조급해하지 말고 점차 보완해 나간다.
- 실제로 내가 직접 필요하여 찾아보고 공부한 것을 기록한다.
- 날짜가 아니라 카테고리로 정리한다.
- 찾기 편하게 분류가 *애매모호*한 게시글의 경우 각 카테고리에 모두 분류해준다.
  - 추후 블로그 개설시 태그(#) 기능을 통해 보완하도록 한다.
