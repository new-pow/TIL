### 체크박스
1. 기본형
```
- [ ] 빈박스
- [x] 체크된 박스

// 주의 : 빈박스도 안에 공백을 넣을 것
```
2. 테이블과 사용하기
```
| Done | To do |
|------|-------|
| - [ ] | Study Markdown |
| - [x] | Study Java |
```

| Done | To do |
|------|-------|
| - [ ] | Study Markdown |
| - [x] | Study Java |
